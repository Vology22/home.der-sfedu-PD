# 🏠 Home.Der — Сервис поиска жилья и сожителей

> Платформа для поиска временного и долгосрочного жилья с системой подбора совместимых сожителей, Telegram-ботом и веб-интерфейсом.

---

## 📌 О проекте

**Home.Der** — это сервис, который помогает пользователям:

- искать жильё;
- находить подходящих соседей;
- получать уведомления о совпадениях;
- управлять анкетой через Telegram и веб-сайт.

Проект объединяет несколько компонентов:

- 🌐 Web Platform
- 🤖 Telegram Bot
- ⚙ Backend API
- 🗄 SQL Database
- 🔄 Синхронизация между ботом и сайтом

---

## 🚀 Основные возможности

### 🤖 Telegram Bot

- Проверка подписки на канал
- Авторизация пользователей
- Создание и редактирование анкеты
- Загрузка фотографий
- Уведомления о новых совпадениях
- Интеграция с WebApp

### 🌐 Веб-сайт

- Просмотр профилей
- Система мэтчей
- Настройки аккаунта
- Профиль пользователя

### ⚙ Backend

- API для взаимодействия сервисов
- Алгоритм подбора пользователей
- Система синхронизации данных
- Работа с SQL-базой данных
